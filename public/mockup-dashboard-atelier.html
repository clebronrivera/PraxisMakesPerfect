Design decisions worth calling out

Each is a deliberate change from the current dashboard. We should land on these before I touch React.

1

Readiness becomes a constellation, not a bar.

A circular arc gives the primary number (42%) real presence and aligns to the engine/orb motif. The flat blue bar is retired.
2

Today's Focus gets one hero — not three peers.

The priority skill glows in its domain color; the next two actions drop to quieter "Then" rows. Visual weight reflects what you should actually do first.
3

Domain bars inherit the atelier palette.

Peach / Mint / Ice / Lavender — the same colors that feed the hero orb now color each domain row. The dashboard and the landing page finally speak the same language.
4

Redemption reads as an eclipsed moon.

Dark card, rose/lavender glow off to the side — quarantined questions in orbit. Reinforces that these items are held, not lost.
5

Feature tiles become "The Toolshed."

Glass cards with color-coded accents and soft hover glows. Each tile borrows from a different domain hue so the toolbar feels like a spectrum, not a box of crayons.
6

Shared language across the app.

Uppercase tracking-wide eyebrows, Inter, the same gradient warm-text for hero numbers, mini-orb in the header. The dashboard now feels like the same product as the landing page.
